# Agency Landing Page

This project is a single-page React application featuring a navigation bar and smooth scrolling to different sections of the page. It serves as the landing page for an agency.

## Features

- **Single Page Application (SPA)**: Ensures a smooth user experience with quick transitions and no page reloads.
- **Responsive Navbar**: A fully responsive navigation bar that provides easy access to all sections of the website.
- **Scroll to Section**: Clicking on navbar items scrolls to the corresponding section smoothly.

## Installation

To get this project up and running on your local machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/NaragintiChanduPriya/agency-landing-page.git
   ```
2. Navigate to the project directory:
   ```bash
   cd agency-landing-page
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

The application will open in your default browser at `http://localhost:3000`.
